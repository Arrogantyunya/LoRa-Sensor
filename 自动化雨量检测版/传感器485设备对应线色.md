# 传感器485设备对应线色

---

## 485雨量传感器（）

### 传感器线色与说明

| 线色 |         说明          |
| :--: | :-------------------: |
| 棕色 | 电源正VCC（12-24VDC） |
| 黑色 |       电源负GND       |
| 黄色 |         485-A         |
| 蓝色 |         485-B         |

### 4芯航插引脚序号对应说明

| 序号 |   说明    |
| :--: | :-------: |
|  1   | 电源负GND |
|  2   |   485-B   |
|  3   |   485-A   |
|  4   | 电源正VCC |

### 实际接线对应线色

| 序号 | 线色 |
| :--: | :--: |
|  1   | 黑色 |
|  2   | 蓝色 |
|  3   | 黄色 |
|  4   | 棕色 |
